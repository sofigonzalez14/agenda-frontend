# Agenda de Tareas – Frontend

Frontend de una mini agenda de tareas con login, pensada para conectarse al backend de **Agenda API** (Node + MySQL).

Permite:
- Iniciar sesión
- Registrarse
- Ver tareas del usuario logueado
- Crear / editar / borrar tareas
- Crear / editar / borrar categorias

## Tecnologías usadas

- **React** – Para la construcción de la interfaz.
- **Vite** – Para el bundling y entorno de desarrollo rápido.
- **Axios / fetch** – Para consumir la API del backend.
- **CSS / estilos custom** – Para la parte visual de la app.
---
Tiene que seguir los mismos pasos que el repo de BACKEND para ejecutarlo.
